# 📊 Habit Tracking Dashboard (Excel Project)

## 🔹 Project Overview

This project is an interactive Habit Tracking Dashboard created using Microsoft Excel. It helps in analyzing daily habits and tracking performance based on completion status (Yes/No).
## Dataset used
- <a href="https://github.com/SK-Tech-Chronicals/Excel-Project-Hub/blob/main/2..png">Dataset View</a>

---

## 🔹 Features

* ✔️ Tracks multiple habits (Study, Exercise, Reading)
* ✔️ Displays Yes vs No comparison
* ✔️ Interactive filters using Slicers
* ✔️ Pivot Table-based analysis
* ✔️ Visual charts for better understanding

---

## 🔹 Dashboard Insights

* Each habit is tracked for 7 days
* Comparison of completed (Yes) vs missed (No) habits
* Helps identify consistency and improvement areas

---

## 🔹 Tools Used

* Microsoft Excel
* Pivot Tables
* Pivot Charts
* Slicers
